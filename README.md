Food App Website

This is a simple food-themed website built using HTML and basic CSS. The project displays different food items in a clean card-style layout and includes a contact section for user interaction.

Features
Food item cards with images and descriptions
Order Now buttons for each item
Simple and clean user interface
Contact form for user details
Responsive layout for different screen sizes

Technologies Used
HTML5
CSS3

Pages and Sections
Home section with food items
Contact section with form

Purpose
This project was created as part of my web development learning journey. It demonstrates my understanding of HTML structure, layout design, and basic CSS styling.

How to Run
Download or clone the repository
Open the index.html file in any web browser
